# Ambience
Ambience is a customizable desktop-like workspace where you can choose content to display and tools to work with. 

It is designed for people who like to work, create or relax with their own visual space.

## The concept
Each feature in Ambience is visually displayed as a window. Just like on a computer. Windows can be added, removed and arranged. By arranging windows users can build their personal workspace. 

<img width="2558" height="1504" alt="image" src="https://github.com/user-attachments/assets/6dbbab73-2004-403f-b563-acb515f5f6c0" />

There are multiple types of [windows](#types-of-windows), which include content, productivity tools and playful blocks.
